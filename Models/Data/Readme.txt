1. После создания таблицы создаем в папке Data Data Transfer Object с именем желаемой таблицы (UserDTO)
2. В web.config добавить строку подключения к БД <connectionStrings>
3. Далее в классе DB создаем контекст данных для созданного DTO при помощи Entity
4. В папке ViewModels создаем визуальную модель для созданного ранее контекста данных UserDTO (UserViewModel),
в котором дублируем поля из таблицы и добавляем промежуточные при необходимости. Там же создаем конструктор, в параметр
которого передаем ранее созданный DTO для связи, и конструктор по умолчанию
5. Создаем контроллер и реализуем задуманные функции
6. После создания нужного метода создаем его визуальное представление. В некоторых случаях создаем частичное представление
7. Метод получения ролей пользователей создается в файлк Global.asax
8. Для корректной работы PayPal меняем локализацию проекта на en в файле Web.config
9. Для создания области Admin ПКМ на проекте -> Добавить область
10. Для обработки запросов связанных с ролями нужно в файле global.asax создать метод Application_AuthenticateRequest
